Minimalist Data Wrangling with PythonÂ¶

This is a draft of Minimalist Data Wrangling with Python by Marek Gagolewski. It is distributed in the hope that it will be useful. Any bug/typos reports/fixes are appreciated. Although available online, this is a whole course, and should be read from the beginning to the end. In particular, refer to the Preface for general introductory remarks. Enjoy.
